# Figures

To generate all pocket figures, open PyMOL and run:

```
run lola_pockets.py
run_all
```

This produces 8 ray-traced PNGs (2400Ã—1800 @ 300 DPI):

- `LprG_TAG_4ZRA_pocket.png`
- `LprG_PIM2_3MHA_pocket.png`
- `VioE_PPA_2ZF4_pocket.png`
- `VioE_PEG_3BMZ_pocket.png`
- `EPD_BCP1_8I34_pocket.png`
- `LppX_2BYO_pocket.png`
- `LolA_2ZPC_pocket.png`
- `RseB_2V42_pocket.png`

Move them into this directory, then rename three for the README gallery:

```bash
cp LprG_TAG_4ZRA_pocket.png figures/lprg_tag_example.png
cp VioE_PEG_3BMZ_pocket.png figures/vioe_peg_example.png
cp EPD_BCP1_8I34_pocket.png figures/epd_bcp1_example.png
```
